# 🚀 InterveuX

A modern full-stack interview preparation and technical assessment platform built to simulate real coding interviews with live collaboration, real-time communication, and secure code execution.

## ✨ Features

- 👨‍💻 Live coding interview environment
- 🎥 Real-time video/audio interaction
- ⚡ Collaborative code editor
- 🔐 Secure authentication system
- 📡 Real-time communication using sockets
- 📊 Automated evaluation workflow
- 🌐 Full-stack architecture
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.IO
